Security_Lab

概要

セキュリティ技術を実践的に学習・検証するためのアプリケーションです。学校の授業やCTF・独学で得た知識をベースに、実際に手を動かして理解を深めることを目的として企画しました。

開発目的

セキュリティは「知識」だけでなく「実際に試す環境」が重要だと感じています。自分専用の検証環境を構築することで、ログ監視・脆弱性検証・インシデント対応の流れを体験的に学ぶことを目指しています。

予定機能

  • ログ監視機能（不審なアクセスの検知・可視化）
  • 疑似攻撃検証機能（安全な環境での攻撃シナリオ再現）
  • 通知機能（アラート発報の仕組み）

使用予定技術

Python / Linux / Docker / AWS
実務に近い構成を意識し、クラウド環境（AWS）上での動作も視野に入れています。AWSについては現在Cloud Practitioner取得に向けて学習中です。

設計方針

実際の企業インフラを想定したアーキテクチャを意識して設計します。コンテナ技術（Docker）で環境を分離し、再現性の高い検証環境を目指します。セキュリティエンジニアとしての視点を持ちながら、インフラ構築のスキルも同時に習得することが狙いです。
